## Codebase Patterns
- Antfarm adds persistent SQLite schema through the centralized migration function in `src/db.ts`; backend persistence changes should follow that path.
- Antfarm backend smoke/integration tests point at built `dist/` modules and isolate SQLite state by setting `ANTFARM_DB_PATH` to a temp database.
- Small backend operator surfaces are exposed through `src/cli/cli.ts` in addition to exported TypeScript functions when a mechanically callable path is needed.
